# SOUL and IDENTITY Customization

`SOUL.md` + `IDENTITY.md` are injected into system prompt every turn.

## SOUL.md should define

- principles
- execution style
- communication tone
- safety/authorization stance

## IDENTITY.md should define

- name
- persona / vibe
- operating temperament

## Practical rule

- If user gives explicit persona requirements, rewrite these files accordingly.
- Keep them concise and operational (not poetic filler only).
