# Skill Library for New Agent

## Minimum set

- `base_skill`
- `memory`
- `cron`

## Common optional set

- `web_skill`
- `code_skill`
- `gateway_skill`
- `todo_skill`
- `knowledge`

## Copy rule

Always copy whole skill directories (including `scripts/` and `references/`), not only `SKILL.md`.

## About `agent-creator`

Copy `agent-creator` only if the new agent should create other agents.
