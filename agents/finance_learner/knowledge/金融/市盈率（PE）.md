# 市盈率（Price to Earnings, P/E）要点

- **定义**：市盈率 = 股价 / 每股收益（EPS）；也可写作 市值 / 净利润。表示市场愿意为公司“1 元利润”支付多少价格。
- **两种常见口径**：
  - **TTM（滚动）**：用最近 12 个月归母净利润或 EPS；更贴近当下。
  - **静态/年报**：用上一财年数据；可能滞后。
  - **Forward P/E（预测）**：用一致预期/预测盈利；对预测误差敏感。
- **解读要点**：
  - P/E 高：可能代表高成长预期/利润短期偏低（周期底部）/估值偏贵。
  - P/E 低：可能代表低成长/风险溢价高/利润处于周期高位（“看似便宜”）。
- **局限与误区**：
  - **亏损公司**：净利润为负时 P/E 无意义或为负，需改用 PS、EV/EBITDA 等。
  - **周期行业**：在盈利高点 P/E 往往偏低、在低点偏高；可看**周期调整**的指标（如 CAPE）或用归一化盈利。
  - **会计口径差异**：一次性损益、资产减值、商誉、非经常性项目会扭曲净利润。
  - **资本结构**：P/E 不显式反映有息负债差异；企业价值类指标（EV/EBITDA）可补充。
- **比较方法**：
  - 与**自身历史区间**比（同公司不同时间）。
  - 与**同业/同商业模式**公司比（横向）。
  - 与**利率/风险溢价**环境结合（利率上行通常压制估值）。

备注：以上为通用金融常识整理；实际计算口径以交易所/数据提供商定义为准。
