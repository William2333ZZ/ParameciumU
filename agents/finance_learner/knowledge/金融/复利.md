# 复利（Compound Interest）要点

- **定义**：利息不只按本金计提，还会对“已产生的利息”再次计息，即“利滚利”。
- **基本公式**：
  - 离散复利：\(FV = PV\,(1+r)^n\)
  - 若每年复利 m 次：\(FV = PV\,(1+\frac{r}{m})^{mn}\)
  - 连续复利：\(FV = PV\,e^{rn}\)
- **关键变量**：
  - \(r\)：每期利率（需与复利频率匹配）；\(n\)：期数；\(m\)：一年复利次数。
- **对比单利**：单利只对本金计息：\(FV = PV\,(1+rn)\)；当 n 增大时，复利增长更快。
- **“72 法则”近似**：年化收益率为 \(r\)（%）时，资产大约在 \(72/r\) 年翻倍（粗略估算，适用于中等利率区间）。
- **实际应用**：定投/养老金/长期持有再投资（股息再投、利息再投）是复利效应的常见来源。
- **注意点**：
  - **名义利率 vs 实际利率**：通胀会侵蚀购买力，实际复利需考虑通胀（近似：\(1+r_{real} \approx (1+r_{nom})/(1+\pi)\)）。
  - **费用与税收**会降低复利的有效增长率。
  - **波动收益**下，“几何平均收益率”更贴近复利长期效果；大幅回撤会显著拖累复利（例如跌 50% 需涨 100% 才回本）。
